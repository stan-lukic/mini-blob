From the solution folder (Linux):
1. run this to trust dev cert:
   dotnet dev-certs https -t
2. Run this test to create container1:
   dotnet test --filter "FullyQualifiedName~AdminCanCreateContainer_CreatesContainerAuthFile"
3. Start the mini blob api: 
   dotnet run --project ./src/MiniBlob.Api/MiniBlob.Api.csproj
